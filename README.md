# アフィリエイトサイト構築用 Claude Code Skills

月2-5万円の収益を目指すアフィリエイトサイトを効率的に構築するためのskill集です。

## 🎯 収益目標
- 月間収益: 2-5万円
- 必要PV: 月間5,000-15,000PV
- 記事数: 30-50記事
- 想定期間: 6-12ヶ月

## 📋 推奨ワークフロー

### 1. 企画・戦略フェーズ (Week 1-2)
```bash
/affiliate-planning      # ジャンル選定、競合分析、収益戦略
/keyword-research        # キーワード調査、検索ボリューム分析
/monetization-strategy   # マネタイズ戦略立案
```

### 2. 設計フェーズ (Week 2-3)
```bash
/site-architecture       # サイト構造設計、カテゴリー設計
/content-calendar        # コンテンツカレンダー作成
/conversion-design       # CV導線設計
```

### 3. 開発・構築フェーズ (Week 3-8)
```bash
/seo-article-generator   # SEO最適化記事の作成
/landing-page-creator    # ランディングページ作成
/internal-linking        # 内部リンク最適化
```

### 4. 最適化フェーズ (Week 8-)
```bash
/seo-audit              # SEO監査・改善提案
/performance-check      # ページ速度最適化
/conversion-optimizer   # CVR改善施策
/analytics-setup        # GA4/Search Console設定
```

## 💡 収益化のポイント

1. **ニッチ選定**: 競合が少なく、購買意欲の高いキーワード
2. **記事品質**: E-E-A-Tを意識した信頼性の高いコンテンツ
3. **内部リンク**: CV記事への適切な導線設計
4. **継続更新**: 最低でも週1-2記事のペース
5. **分析改善**: GA4でユーザー行動を分析し継続改善

## 📊 成功指標

- 検索順位: 狙ったキーワードでTop10入り
- クリック率: 検索結果で3-5%以上
- 直帰率: 60%以下
- 平均セッション時間: 2分以上
- コンバージョン率: 1-3%

## 🔧 推奨ツールスタック

- **CMS**: Blogger (完全無料) → 収益安定後にWordPress移行も検討
- **SEOツール**: Google Search Console, ラッコキーワード (無料)
- **アナリティクス**: Google Analytics 4, Microsoft Clarity (無料)
- **ASP**: A8.net, もしもアフィリエイト, バリューコマース
- **初期費用**: ¥0 (ドメイン・サーバー不要)
