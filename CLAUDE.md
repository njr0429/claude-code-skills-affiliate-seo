# アフィリエイトサイト構築 Claude Code Skills

月2-5万円の収益を目指すアフィリエイトサイトを効率的に構築するためのClaude Code専用スキル集

---

## 🎯 プロジェクト目的

初心者でも**月2-5万円の収益**を達成できるアフィリエイトサイトを構築する

### 目標指標
- 月間収益: ¥20,000-50,000
- 必要PV: 5,000-15,000/月
- 記事数: 30-50記事
- 達成期間: 6-12ヶ月
- 初期費用: ¥0（Blogger利用）

---

## 📐 基本方針

### プラットフォーム戦略
```
Phase 1: Blogger（完全無料）
  ↓ 収益が月3-5万円安定
Phase 2: 独自ドメイン追加（¥1,000/年）
  ↓ さらに収益拡大
Phase 3: WordPress移行検討（月¥1,000）
```

**原則**: リスクゼロで始め、収益が出てから投資する

### プラットフォーム別対応

#### Blogger（デフォルト：無料スタート）
```
✅ シンプルHTML（インラインCSS）
✅ ラベル運用（カテゴリーの代わり）
✅ URL制御は限定的
✅ 手動貼り付け前提
✅ 画像はBlogger標準機能で挿入
❌ 複雑なJavaScript不可
❌ カスタムフィールド不可
```

#### WordPress（移行後：収益安定後）
```
✅ shortcode/プラグイン活用
✅ カテゴリー・パーマリンク最適化
✅ カスタムCSS/JavaScript
✅ 比較表やCTAの再利用性高い
✅ Yoast SEOなどプラグイン利用可能
```

**各スキル共通**: 特に指定がなければ**Blogger前提**で出力します

### SEO方針

#### 1. ターゲット選定
- ✅ ロングテールキーワード（3-4語）
- ✅ 月間検索100-10,000
- ✅ 競合DR30未満
- ❌ ビッグキーワード（月間10万以上）避ける
- ❌ YMYLジャンル（医療・金融・法律）厳禁

#### 2. コンテンツ品質
- E-E-A-T準拠必須
  - **Experience**: 実体験・使用レビュー
  - **Expertise**: 専門知識・データ
  - **Authoritativeness**: 信頼できる情報源
  - **Trustworthiness**: 正確で偏りのない情報

#### 3. 内部リンク戦略
- トピッククラスター構造
- 収益記事に最低5個の被リンク
- 孤立ページゼロ

## 🗣️ 言語・トーン

### 言語
- 日本語
- です・ます調

### トーン
- 親しみやすく、堅苦しくない
- 専門的すぎない（初心者が理解できる）
- 実践的・具体的
- 誠実で信頼できる

### 避けるべき表現
- ❌ 過度な煽り（「今すぐ！」「絶対！」）
- ❌ 誇張表現（「確実に稼げる」）
- ❌ 専門用語の乱用
- ❌ 断定的すぎる表現

---

## 🚫 禁止事項

### 絶対に避けるもの
1. **YMYLジャンル**
   - 医療、健康、金融、法律は扱わない
   - Googleのガイドライン違反リスク

2. **虚偽・誇張**
   - 嘘の情報、誇張した表現
   - 存在しないキャンペーン情報
   - 偽の口コミ・レビュー

3. **スパム的SEO**
   - キーワード詰め込み
   - 隠しテキスト
   - 自動生成コンテンツそのまま
   - 低品質な被リンク購入

4. **著作権侵害**
   - 他サイトのコピペ
   - 無断転載
   - 画像の無断使用

## 📏 記事品質基準（最低ライン）

### 必須要件
```
✅ 文字数: 2,000文字以上
✅ 画像: 5枚以上（alt属性必須）
✅ 見出し: H2が3つ以上
✅ オリジナル情報: 必ず含める
✅ 実体験: 可能な限り含める
✅ 内部リンク: 3-5箇所
✅ 外部リンク: 信頼できるソース1つ以上
```

### 推奨要件
```
⭐ 文字数: 競合平均+10-15%
⭐ 画像: オリジナル写真3枚以上
⭐ 図解・表: 視覚的にわかりやすく
⭐ 比較表: 商品比較がある場合必須
⭐ 口コミ: 良い・悪い両方を掲載
```

### 収益記事の追加要件
```
💰 CTAボタン: 3箇所以上
💰 アフィリエイトリンク: 自然な文脈で2-3箇所
💰 商品比較: 最低3商品
💰 購入導線: 明確で自然
```
