# SEO Article Generator Skill

**目的**: 検索上位を狙えるSEO最適化された高品質記事を効率的に作成する

## このスキルを使うタイミング
- キーワードリサーチ完了後
- 新規記事を作成するとき
- 既存記事をリライトするとき

## 実行内容

### 1. 記事企画ヒアリング
ユーザーに以下を質問:
- 対象キーワード
- 記事タイプ (収益記事 / 集客記事)
- 想定文字数 (2,000-8,000文字)
- アフィリエイト案件 (収益記事の場合)

### 2. 競合記事分析
上位10記事を分析:
- 見出し構成
- 文字数
- 独自コンテンツ
- 不足している情報

### 3. 記事構成案作成

```markdown
# [タイトル (30-40文字、キーワード含む)]

## メタディスクリプション (120文字以内)
[検索結果に表示される説明文]

## 目次
1. [導入]
2. [H2見出し1] - キーワードの基礎知識
3. [H2見出し2] - 具体的な方法・手順
4. [H2見出し3] - おすすめ商品・サービス ★収益ポイント
5. [H2見出し4] - よくある質問
6. [まとめ]

内部リンク挿入箇所: 3箇所
アフィリエイトリンク: 2-3箇所
画像: 5枚以上
```

### 4. 記事本文生成

#### E-E-A-T を意識した執筆
- **Experience (経験)**: 実体験・実例を含める
- **Expertise (専門性)**: 専門的な知識を示す
- **Authoritativeness (権威性)**: 信頼できる情報源を引用
- **Trustworthiness (信頼性)**: 正確で偏りのない情報

#### SEO最適化ポイント
```
✅ タイトルタグ: 主要キーワード前方配置、30-40文字
✅ H1タグ: タイトルと同じ
✅ H2-H3: 関連キーワード自然に含める
✅ 最初の100文字: キーワード含める
✅ キーワード出現率: 1-2%程度
✅ 共起語: 関連語を自然に散りばめる
✅ 文字数: 競合の平均+10%以上
✅ 内部リンク: 3-5箇所
✅ 外部リンク: 公式サイト、統計データなど
✅ 画像alt属性: キーワード含める
```

#### 収益記事の場合
```markdown
## おすすめ[商品カテゴリ]TOP3

### 1位: [商品名]
[商品画像]

**おすすめポイント**
- ポイント1

- ポイント3

**こんな人におすすめ**
- タイプ1
- タイプ2

**口コミ・評判**
> [実際の口コミ引用]

[アフィリエイトボタン]

---

### 2位: [商品名]
...
```

### 5. コンバージョン導線設計

記事内のCV導線:
1. **導入部分**: 問題提起 → 解決策の提示
2. **中盤**: 詳細情報 → 比較表・ランキング
3. **終盤**: CTA (Call to Action)

```html
<!-- CTAボタン例 -->
<div class="cta-box">
  <p class="cta-text">今すぐ公式サイトで詳細をチェック！</p>
  <a href="[アフィリエイトリンク]" class="cta-button" target="_blank" rel="nofollow">
    公式サイトはこちら
  </a>
</div>
```

### 6. 品質チェックリスト

記事完成後、以下を確認:
- [ ] タイトルにキーワードが含まれている
- [ ] メタディスクリプションが120文字以内
- [ ] H2見出しが3つ以上ある
- [ ] 文字数が2,000文字以上
- [ ] 画像が3枚以上挿入されている
- [ ] 内部リンクが3箇所以上ある
- [ ] 外部リンクが1つ以上ある
- [ ] 誤字脱字がない
- [ ] スマホで読みやすい (短い段落)
- [ ] オリジナル性がある (コピペではない)

## 記事タイプ別テンプレート

### 商標キーワード記事
```
タイトル例: 「[商品名]の口コミ・評判は?実際に使った感想を正直レビュー」

構成:
1. [商品名]とは
2. [商品名]の特徴・スペック
3. 実際に使った口コミ・評判
4. メリット・デメリット
5. こんな人におすすめ
6. 最安値で購入する方法
7. よくある質問
8. まとめ
```

### 比較・ランキング記事
```
タイトル例: 「[カテゴリ]おすすめ10選!目的別ランキングと選び方」

構成:
1. [カテゴリ]の選び方
2. おすすめランキングTOP10
3. 目的別おすすめ
4. 比較表
5. よくある質問
6. まとめ
```

### HowTo記事 (集客)
```
タイトル例: 「[〇〇のやり方]初心者でも5分でできる簡単ガイド」

構成:
1. [〇〇]とは
2. 準備するもの
3. 手順1
4. 手順2
5. 手順3
6. よくある失敗と対処法
7. さらに詳しく学ぶなら (内部リンク)
8. まとめ
```

## 成功のポイント
- ✅ 競合を超える情報量と独自性
- ✅ 読者の検索意図を120%満たす
- ✅ 体験談・実例で差別化
- ✅ 図解・表で視覚的にわかりやすく
- ✅ モバイルファーストで読みやすく
