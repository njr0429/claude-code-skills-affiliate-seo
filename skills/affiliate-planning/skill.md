# Affiliate Planning Skill

## 概要
アフィリエイトサイトの戦略立案、ジャンル選定、競合分析を行い、月2-5万円の収益目標を達成するための基礎を固める

---

## Inputs（入力）

### 必須
なし（対話形式でヒアリング）

### ヒアリング項目
実行時に以下をユーザーに質問:
- `interest_field`: 興味のある分野・得意分野
- `work_hours`: 月間作業時間の目安（週何時間）
- `initial_budget`: 初期投資可能額（¥0 = Blogger / ¥5,000-10,000 = WordPress）
- `target_period`: 目標達成期間（何ヶ月）
- `domain_plan`: 将来的な独自ドメイン取得の予定（有/無）

---

## Outputs（出力）

### 形式
Markdown形式の企画書

### 含まれる内容
1. サイトテーマ（選定したジャンル）
2. ターゲット読者
3. プラットフォーム選定（Blogger/WordPress）
4. 収益戦略（ASP、案件、予想CVR）
5. 競合分析サマリー（上位10サイト）
6. キーワード戦略（収益記事5-10個、集客記事20-40個）
7. 6ヶ月ロードマップ
8. 初期記事リスト（10記事）

### サンプル出力
```markdown

## 次のステップ
1. /keyword-research で詳細キーワード調査
```

## このスキルを使うタイミング

### ✅ 使うべきとき
- 新規アフィリエイトサイトを立ち上げる前（最初のステップ）
- ジャンル選定に迷っているとき
- 収益性の高いニッチを見つけたいとき
- 複数のジャンル候補があり、どれを選ぶか決めたいとき
- 既存サイトのジャンル変更・ピボットを検討しているとき

### ❌ 使わない方がいいとき
- すでにジャンルが確定していて、記事作成を始める段階
- 既存サイトで記事を追加する段階（キーワードリサーチへ）
- ジャンル選定ではなく、具体的な記事構成を考えたいとき

### 📍 ワークフロー上の位置
```
（開始）→ このスキル → /keyword-research → /site-architecture
```

## 実行手順

### 1. ユーザーヒアリング
以下の項目をユーザーに質問:
- 興味のある分野・得意分野
- 月間作業時間の目安
- 初期投資可能額 (Blogger利用で¥0から可能)
- 目標達成期間
- 将来的な独自ドメイン取得の予定

### 2. ジャンル候補の提案
以下の基準でジャンルを評価:
- ✅ 検索需要: 月間検索ボリューム1,000以上
- ✅ 競合難易度: ドメインパワー30未満のサイトが上位にいる
- ✅ 収益性: 成果報酬1,000円以上のASP案件が存在
- ✅ 継続性: トレンドではなく安定した需要

### 3. 競合分析
選定ジャンルの上位10サイトを分析:
- ドメインパワー
- 記事数
- 更新頻度
- コンテンツの質
- 収益化手法

### 4. 収益シミュレーション
```
月間目標収益: 2-5万円
必要CV数: 20-50件 (単価1,000円の場合)
必要PV数: 5,000-15,000PV (CVR1-3%想定)
必要記事数: 30-50記事
```

### 5. アクションプラン策定
- サイトテーマの決定
- 主要キーワードリスト (10-20個)
- 初期記事リスト (10記事)
- 6ヶ月間のマイルストーン
